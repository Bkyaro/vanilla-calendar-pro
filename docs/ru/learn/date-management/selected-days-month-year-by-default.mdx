# Выбранные дни, месяц, год по умолчанию

Календарь позволяет явно указать выбранные дни по умолчанию, а также месяц и год, которые будут отображаться независимо от текущей даты.

Пример ниже демонстрирует установку выбранных дней, месяца и года по умолчанию.

Это полезно, если вам необходимо предварительно выбрать определенные даты и установить определенный месяц и год при инициализации календаря.
