# Поп-апы

Календарь позволяет добавлять поп-апы с информацией для любого дня, которые будут отображаться при наведении на этот день.

Это полезное средство для отображения дополнительных данных в календаре.

В приведенном примере выделены определенные дни с использованием CSS-модификаторов, и для каждого из них добавлена информация в поп-апах.

Дополнительные детали о поп-апах можно найти в <a href="/docs/reference/additionally/pop-ups">справочнике</a>.
