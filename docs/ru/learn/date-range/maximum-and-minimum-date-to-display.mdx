# Максимальная и минимальная дата для отображения

Параметры `settings.range.min` и `settings.range.max` определяют диапазон дат, которые можно отображать в календаре, но не влияют на жизненный цикл календаря. Они лишь указывают, какие даты разрешено отображать и выбирать.

Например, если параметр `settings.visibility.disabled` установлен в значение `true`, то минимальный и максимальный год, доступные для просмотра пользователем, будут определены значениями параметров `date.min` и `date.max`.
